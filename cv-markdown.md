## Grants and Funding

*Current Funding, which includes only amounts over 10000 USD*

`01/2021-12/2023.` **Principal Investigator**, *Brain-Computer Interface to Exoskeleton System Design for Personalized Upper-Limb Rehabilitation after Stroke*, Nazarbayev University.

This project aims to accelerate the efficacy of robotic rehabilitation using a new neural interface system developed for upper-limb repair after stroke. Its goals are to facilitate motor recovery from incomplete upper-limb paralysis and to investigate the cortical reorganization in the brain associated with the recovery process.

`01/2021-12/2023.` **Co-Investigator**, *Classifier Validity and Its Applications in Bioinformatics*, Nazarbayev University.

This project focuses on studying measures of classifier validity and exploring their applications in bioinformatics.

`01/2019-12/2021.` **Co-Principal Investigator**, *Neural-assisted Optimization of Human Learning with Multimodal Neuroimaging of EEG and NIRS*, Nazarbayev University.

The objective of this project is to implement and validate a neural-assisted, machine-learning-based intelligent tutoring system. This system utilizes information from physiological signals to adapt the learning schedule to the individual learning capacity of the user.

This project focuses on understanding the neurophysiological correlates of motor learning during movements in stroke patients.
